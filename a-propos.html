À propos

Une veille claire pour les métiers du génie civil

Génie Civil Actu rassemble des titres, résumés courts et liens vers les sources originales afin de faciliter la veille technique.

Objectif

Offrir un point d’entrée simple pour suivre les sujets liés aux structures, routes, ponts, matériaux, BIM, hydraulique, géotechnique et chantiers.

Fonctionnement

Le site lit un fichier JSON local généré par un script Python et mis à jour automatiquement par GitHub Actions.

Choix technique

Le projet reste entièrement statique : HTML, CSS, JavaScript simple, Python pour l’agrégation et GitHub Pages pour l’hébergement.

Respect des sources

Le site ne publie pas les articles complets et ne télécharge pas les images. Chaque actualité renvoie vers la source originale.
