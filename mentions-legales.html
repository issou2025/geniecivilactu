Mentions légales

Attribution, responsabilité et respect des sources

Cette page précise les règles appliquées par Génie Civil Actu pour présenter les actualités sans recopier les contenus originaux.

Contenus affichés

Le site affiche uniquement des titres, de courts résumés, des dates, des catégories, des sources et des liens vers les articles originaux.

Droits d’auteur

Les articles complets, images, marques et contenus appartiennent à leurs auteurs, médias et éditeurs respectifs.

Images

La version actuelle ne télécharge pas les images des sources. Un visuel neutre est affiché lorsqu’aucune image autorisée n’est fournie.

Responsabilité

Génie Civil Actu est un outil de veille. Les informations techniques doivent être vérifiées auprès des sources officielles et des documents professionnels applicables.

Données personnelles

Le site n’utilise pas de compte utilisateur, ne possède pas de base de données et ne stocke pas les messages du formulaire de contact.

Automatisation

Les actualités sont mises à jour par un script Python lancé avec GitHub Actions. Une source indisponible n’empêche pas les autres sources d’être traitées.
